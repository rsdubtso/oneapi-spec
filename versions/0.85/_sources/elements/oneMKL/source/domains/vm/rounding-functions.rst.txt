Rounding Functions
==================

.. container:: familylinks


   .. container:: parentlink


      **Parent topic:**Â `VM Mathematical
      Functions <vm-mathematical-functions.html>`__


.. container::


   -  `floor <floor.html>`__
      Computes an integer value rounded towards minus infinity for each
      vector element.
   -  `ceil <ceil.html>`__
      Computes an integer value rounded towards plus infinity for each
      vector element.
   -  `trunc <trunc.html>`__
      Computes an integer value rounded towards zero for each vector
      element.
   -  `round <round.html>`__
      Computes a value rounded to the nearest integer for each vector
      element.
   -  `nearbyint <nearbyint.html>`__
      Computes a rounded integer value in the current rounding mode for
      each vector element.
   -  `rint <rint.html>`__
      Computes a rounded integer value in the current rounding mode.
   -  `modf <modf.html>`__
      Computes a truncated integer value and the remaining fraction part
      for each vector element.
   -  `frac <frac.html>`__
      Computes a signed fractional part for each vector element.
