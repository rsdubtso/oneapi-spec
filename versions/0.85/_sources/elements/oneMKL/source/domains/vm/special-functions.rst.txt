Special Functions
=================

.. container:: familylinks


   .. container:: parentlink


      **Parent topic:**Â `VM Mathematical

.. container::


   -  `erf <erf.html>`__
      Computes the error function value of vector elements.
   -  `erfc <erfc.html>`__
      Computes the complementary error function value of vector
      elements.
   -  `cdfnorm <cdfnorm.html>`__
      Computes the cumulative normal distribution function values of
      vector elements.
   -  `erfinv <erfinv.html>`__
      Computes inverse error function value of vector elements.
   -  `erfcinv <erfcinv.html>`__
      Computes the inverse complementary error function value of vector
      elements.
   -  `cdfnorminv <cdfnorminv.html>`__
      Computes the inverse cumulative normal distribution function
      values of vector elements.
   -  `lgamma <lgamma.html>`__
      Computes the natural logarithm of the absolute value of gamma
      function for vector elements.
   -  `tgamma <tgamma.html>`__
      Computes the gamma function of vector elements.
   -  `expint1 <expint1.html>`__
      Computes the exponential integral of vector elements.
