
atanh
=====


.. container::


   Computes inverse hyperbolic tangent of vector elements.

   Notes:


   -  ``atanh(±1±i*0)=±∞±i*0``, and status::sing error is generated


   -  ``atanh(CONJ(a))=CONJ(atanh(a))``


   -  ``atanh(-a)=-atanh(a)``.
